快速设置

设置（Settings） 是关卡编辑器中用于控制选择、编辑和预览等方面的一组属性。你可以在关卡编辑器工具栏中访问它们。

选项

说明

选择

允许半透明选择（Allow Translucent Selection）

勾选后，单击应用了半透明材质的几何体将选择Actor。

启用该选项可以选择透明网格体，如玻璃物体。禁用该选项在其他情况下或许非常有用，例如在一个充满半透明粒子的房间里。由于单击粒子就会选中发射器Actor，选择房间里的其他对象会非常困难。

允许群组选择（Allow Group Selection）

勾选后，选择一个群组中的Actor就会选择这个群组，而不是单个Actor。

严格框选择（Strict Box Selection）

勾选后，Actor必须完全位于选取框中才能被选中。

框选择遮挡对象（Box Select Occluded Objects）

勾选后，则选取框选择操作也会选择被其他对象遮挡的对象。

显示变换控件（Show Transform Widget）

切换视口中的变换控件的可视性。

可延展性

引擎可延展性设置（Engine Scalability Settings）

允许快速访问用来控制各种渲染功能保真度的可延展性设置：

材质质量级别（Material Quality level）

设置视口中用于预览的材质质量级别。如需进一步了解关于如何设置材质以和本设置一起使用，请参阅材质质量切换表达式。

质量级别

低

高

预览渲染级别（Preview Rendering Level）

设置编辑器使用的渲染级别。

实时音频

音量（Volume）

控制视口设置为实时时场景中播放的音量。

对齐

启用Actor对齐（Enable Actor Snapping）

勾选后，如果位于指定距离内，Actor会对齐其他Actor的位置。

距离（Distance）

勾选了 启用Actor对齐（Enable Actor Snapping） 后，设置Actor之间要进行对齐至少必须达到的距离。

启用插槽对齐（Enable Socket Snapping）

勾选后，允许Actor与插槽对齐。

启用顶点对齐（Enable Vertex Snapping）

勾选后，Actor将沿着移动方向对齐另一个Actor的最近顶点。

启用平面对齐（Enable Planar Snapping）

勾选后，Actor将对齐约束平面上的最近位置。此功能仅在透视视图中才能正常使用。

视口

隐藏视口UI（Hide Viewport UI）

切换隐藏的视口UI模式，隐藏所有覆盖的视口UI控件。

预览

绘制笔刷多边形（Draw Brush Polys）

勾选后，将对选中的CSG笔刷面部渲染半透明多边形。

启用

禁用

在游戏预览中仅加载可见关卡（Only Load Visible Levels in Game Preview）

勾选后，在游戏预览开始时仅加载可见关卡。

启用粒子系统LOD切换（Enable Particle System LOD Switching）

勾选后，粒子系统将在透视图视口中使用距离LOD切换。

切换粒子系统助手（Toggle Particle System Helpers）

在视口中切换显示粒子系统助手。

冻结粒子模拟（Freeze Particle Simulation）

勾选后，粒子系统将冻结其模拟状态。

启用LOD视图锁定（Enable LOD View Locking）

勾选后，同类型视口将使用同一LOD。

启用自动关卡流送（Enable Automatic Level Streaming）

勾选后，视口将在摄像机移动时自动流送关卡。
