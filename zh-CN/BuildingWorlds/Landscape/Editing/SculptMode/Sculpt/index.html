通过 造型（Sculpt） 工具能以当前选中笔刷和衰减的形状升高和降低地形高度图的高度。

使用造型工具

在此例中，造型工具应用高度数值升高或降低地形的高度数据。 在需要进行绘制的区域上点击并移动即可基于使用的工具设置调整数值。

可使用以下功能键打造地形高度图：

功能键

操作

鼠标左键

增加高度图或所选图层的权重。

Shift + 鼠标左键

减小高度图或所选图层的权重。

工具设置

属性

描述

Tool Strength

设定每次笔刷笔划效果的量。如您使用的是压力感应的笔/平板，这将会影响工具的强度。

Use Target Value

启用每次笔刷笔划工具进行混合的目标值。

笔刷强度决定使用造型工具时升高或降低的量。
