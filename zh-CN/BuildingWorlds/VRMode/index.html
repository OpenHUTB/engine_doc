虚幻引擎VR模式

使用编辑器工具集的完整功能，结合专为VR设计的互动，在虚拟现实环境中构建世界场景。

VR模式可使用编辑器工具集的完整功能，结合专为虚拟现实（VR）世界构建设计的互动模型，即可在虚拟现实环境中设计并构建世界场景。

直接在VR中工作可获得良好的规模感，有助于创建无比真实的世界场景，而使用运动控制器可打造出包含自然运动和交互的环境。

要求

开发环境满足以下配置后，方可在VR模式下使用虚幻编辑器：

硬件要求

PC 硬件

VR硬件的最低要求

支持的VR硬件

  • Oculus Rift及Oculus触摸控制器

  • HTC Vive及控制器

VR编辑器当前需要运动控制器，无法单独使用头显。

软件要求

操作系统

Windows 7/8 64 位（或更高版本）

SDK

  • Oculus Rift：Oculus Runtime 1.3或更新版本

  • HTC Vive：SteamVR

显卡驱动

必须安装最新显卡驱动。部分HMD功能需要最新驱动。
