ChunkDownloader 补丁系统是 虚幻引擎 的内置系统，提供了为游戏打补丁的功能。本指南逐步介绍如何设置插件，让你可以在游戏代码中使用它。

必要设置

在使用 C++ 项目时才能使用 ChunkDownloader。如果你正在使用蓝图项目，则可以创建新的C++类，以将其转换成C++项目。

步骤

  1. 打开你的 项目设置（Project Settings），浏览至 项目（Project） > 打包（Packaging），然后确保 使用Pak文件（Use Pak File） 和 生成文件块（Generate Chunks） 均已启用。

  2. 打开插件窗口并启用 Chunk Downloader 插件。重启编辑器，使你的更改生效。

  3. 在Visual Studio中，打开你项目的 Build.cs 文件，并将 ChunkDownloader 作为私有依赖性模块名称添加到你的ModuleRules。

    PrivateDependencyModuleNames.AddRange(

    new string[]
{
    "ChunkDownloader"
}

  4. 将你的更改 保存（Save） 到这些文件。

  5. 右键点击你的 .uproject 文件，然后点击 生成项目文件（Generate Project Files）。

  6. 返回到你在 Visual Studio 中的项目解决方案，然后 构建（build） 项目。

最终结果

现在ChunkDownloader已经可以在你的项目中使用，你可以在游戏代码中接着实现它们，以便下载和安装.pak文件。
