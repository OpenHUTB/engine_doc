Google Daydream Performance HUD

Daydream Performance HUD 是一个性能显示层，在所有以 Google VR SDK 1.60 及以上版本编译的应用程序中均启用。

Google Daydream 自带内置性能显示层，启用后即可迅速找到性能问题的缘由。启用此显示层后将显示以下数据：

  • 应用程序帧提交率，以及帧是否被阻塞在 GPU 上。

  • 异步二次投影统计数据。

  • 处理内存使用情况。

  • 设备距离热量节流的温度（以摄氏度为单位）。

启用 Performance HUD

进行以下操作即可启用 Performance HUD。

  1. 运行 Daydream 应用程序后，点击屏幕右上角的齿轮图标打开 VR 设置。

  2. 点击 版本号 选项 6 次，使 Developer Options 出现。

    点击查看全图。

  3. 选择 Developer Options 然后点击 Enable performance heads-up display 旁的复选框启用它。启用此项后也将启用 Enable Performance Monitoring 选项。

    点击查看全图。

  4. 下次运行 VR 应用程序时便会显示性能显示层，如下图所示：

    点击查看全图。

    如需了解 Performance HUD 使用的更多信息，请查阅官方 Google VR Performance HUD 文档。
