安装额外的Android SDK

本页面的内容

你可能需要安装额外的Android SDK才能使你的虚幻引擎4项目能够正常运行，具体取决于你正在针对哪种类型的Android设备进行开发。在以下文档中，我们将为你介绍如何安装额外的Android SDK。

步骤

要安装额外的Android SDK，需要首先执行以下步骤打开 Android SDK Manager。

  1. 首先，在PC上找到 NVPACK\android-sdk-windows\tools 文件夹。

  2. 找到 Android.bat 文件，然后双击将其打开。

  3. Android SDK Manager 加载后，按 Deselect All 选项来确保不会卸载任何可能需要的SDK。

  4. 接下来，找到要安装的Android API版本并展开可用的选项。在此示例中，我们将使用 Android 7.1.1（API 25），因此请确保在"Android 7.1.1（API 25）"部分下面选中 SDK Platform 旁边的复选框。

  5. 最后，按 Install 1 package 按钮以开始下载和安装。

最终结果

完成后，Status 处将变为上面带有勾号的文件夹图标，表示该版本已安装。

步骤

要安装额外的Android SDK，需要首先执行以下步骤打开 Android SDK Manager。

  1. 首先，在Mac上找到 NVPACK\android-sdk-macosx\tools 文件夹。

  2. 找到 Android 文件，然后双击以将其打开。

    如果在尝试启动Android更新程序时收到以下错误，需要从苹果网站上安装以下版本的Java。

    • Java for OS X 2017-001

  3. Android SDK Manager 加载后，按 Deselect All 选项来确保不会卸载任何可能需要的SDK。

  4. 接下来，找到要安装的Android API版本并选中 SDK Platform 旁边的复选框来对它进行标记以下载和安装。

  5. 最后，按 Install 1 package 按钮以开始下载和安装。

最终结果

完成后，Status 处将变为上面带有勾号的文件夹图标，表示该版本已安装。
