这些模块把旋转恒定速率应用到粒子上。

Init Mesh RotRate

Initial RotRate 模块用于设置网格体粒子被发射时的旋转速率。它包含下列属性：

属性

描述

Rotation

Start Rotation Rate

一个浮点 distribution，用于确定粒子所拥有的旋转速率（以每秒内的旋转次数为单位）。基于粒子生成的 EmitterTime 获取该数值，并将其添加至 Particle RotationRate 和 Base RotationRate 数值。

Init Mesh RotRate (Seed)

Initial RotRate (Seeded) 模块与发射时设置网格体粒子旋转速率的 Init Mesh RotRate 模块相同；然而此模块可对种子信息进行指定。该种子信息用于选择 distribution 值，从发射器每次使用的模块获得更稳定的效果。它包含下列属性：

属性

描述

RandomSeed

Random Seed Info

为该模块属性选择"随机"值而使用的随机种子。

属性

描述

Get Seed From Instance

如为 ture，则模块将尝试从实例所有者处获得种子（Seed）。如获取失败，将返回从 Random Seeds 阵列中获得种子。

Instance Seed Is Index

如为 ture，从实例中恢复的种子值即为 Random Seeds 阵列中的索引。

Parameter Name

设置种子所放置实例显示的名称。

Random Seeds

本模块应用的随机种子值。如多个值被指定，实例将随机选取一个值。

Reset Seed On Emitter Looping

如为 ture，每次发射器循环时将重新设置该种子。

Rotation

Start Rotation Rate

一个浮点 distribution，用于确定粒子所拥有的旋转速率（以每秒内的旋转次数为单位）。基于粒子生成的 EmitterTime 获取该数值，并将其添加至 Particle RotationRate 和 Base RotationRate 数值。

Mesh RotRate*Life

Mesh RotRate Life 模块用于调整网格体粒子在其生命周期中的旋转速率。它包含下列属性：

属性

描述

Rotation

Life Multiplier

一个浮点 distribution，用于确定应用于粒子旋转速率的缩放因子。基于粒子生成和更新时的 RelativeTime 获取该数值，且该数值将乘以 Particle.RotationRate 数值。

Mesh Rot Rate Over Life

Mesh Rot Rate Over Life 模块用于设置网格体粒子在其生命周期中的旋转速率。它包含下列属性：

属性

描述

Rotation

Rot Rate

一个矢量 distribution，用于确定粒子所拥有的旋转速率（以每秒内的旋转次数为单位）。使用粒子相对时间获取数值。

Scale Rot Rate

如为 true，粒子的当前旋转速率将根据 Rot Rate distribution 数值进行调整。
