Remove 节点将从 Map 移除一个键值对，从而减少 Map 的长度。从 Map 移除键值对时，节点将检查键是否在 Map 中。 如键未在 Map 中，键值对将不会被移除。

输入

引脚位置

名称

描述

(In) Exec

输入执行引脚。

Target Map

需要移除键值对的 Map。

Key

用于在 Map 中查找值的键。

键必须包含在 Map 中，键值对才能被移除。

输出

引脚位置

名称

描述

(Out) Exec

输出执行引脚。

Boolean Return Value

如键值对被移除，节点则返回 true。如 Map 中的值不使用提供的键，节点则返回 false。
