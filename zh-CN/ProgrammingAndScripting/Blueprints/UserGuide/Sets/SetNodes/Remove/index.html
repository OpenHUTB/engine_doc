Remove 节点将一个项目从 Set 中移除，从而缩短 Set 的长度。将项目从 Set 中移除时，节点将检查项目是否在 Set 中。 如项目未在 Map 中，则其将不会被移除。

输入

引脚位置

名称

描述

(In) Exec

输入执行引脚。

Target Set

需要移除项目的 Set。

New Item

需要从 Set 中移除的项目。

项目从 Set 中被移除前，它必须实际存在。

输出

引脚位置

名称

描述

(Out) Exec

输出执行引脚。

Boolean Return Value

如项目被移除，此节点将返回 true。如未在 set 中找到相同项目，此节点则会返回 false。
