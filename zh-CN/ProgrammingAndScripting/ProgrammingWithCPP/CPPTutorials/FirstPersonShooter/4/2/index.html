4.2 - 设置事件图表

学习如何设置第一人称射击游戏角色的事件图表。

Beginner

Choose your operating system:

Windows

macOS

Linux

本页面的内容

  • 更新状态变量

  • 查询角色移动

  • 确定角色是否处于下落状态

  • 确定角色是否处于奔跑状态

在这步中，我们将编辑动画的事件图表，确保游戏运行时动画转换变量正确发出。

更新状态变量

  1. 双击 My Blueprint 标签中的 EventGraph 打开事件图表。

  2. 在图表中单击右键呼出 快捷菜单。

  3. 在 快捷菜单 搜索域中输入"Update"，然后点击 Event Blueprint Update Animation 添加该节点。

    使用 Event Blueprint Update Animation 节点可在动画更新时对状态变量进行更新，使它们固定与游戏状态同步。

  4. 在图表中单击右键呼出 快捷菜单。

  5. 在 快捷菜单 搜索域中输入"Owner"，然后点击 Try Get Pawn Owner 添加该节点。

  6. 从输出引脚连出引线，并在 快捷菜单 中选择 Cast to Character。

  7. 将 Event Blueprint Update Animation 输出执行引脚和 Cast to Character 输入执行引脚用引线连接起来。

  8. 从 As Character 输出引脚连出引线并选择 Get Character Movement。

  9. 从 Character Movement 输出引脚连出引线并选择 Get Movement Mode。

查询角色移动

  1. 从 Movement Mode 输出引脚连出引线并选择 Equal (Enum)。

确定角色是否处于下落状态

  1. 将 Equal (Enum) 节点上的下拉值设为 Falling。

  2. 按住 Alt 键点击 My Blueprint 标签中的 IsFalling，拖入图表创建一个 Set Is Falling 节点。

  3. 将 Cast to Character 节点未标记的输出执行引脚和 Set Is Falling 的输入执行引脚相连；将 Equal (Enum) 节点的输出布尔数据引脚和 Set Is Falling 节点的输入布尔数据引脚相连。

确定角色是否处于奔跑状态

  1. 返回 Cast To Character 节点，再次从 As Character 引脚连出引线。选择 Get Velocity 节点。

  2. 如角色不为站立状态，其速度矢量的长度将大于零。因此，从 Return Value 矢量输出引脚连出引线并选择 Vector Length，将此节点添加到图表。

  3. 从 Return Value 浮点输出引脚连出引线并选择 > (float) 节点。

  4. 按住 Alt 键点击 My Blueprint 标签中的 IsRunning，拖入图表创建一个 Set Is Running 节点。

  5. 将 Set Is Falling 节点的输出执行引脚和 Set Is Running 的输入执行引脚相连；将 > (float) 节点的输出布尔数据引脚和 Set Is Running 节点的输入布尔数据引脚相连。
