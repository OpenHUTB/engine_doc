可从外部图像编辑程序（如 Photoshop 或 GIMP）将纹理导入虚幻引擎。我们在此使用 Photoshop 进行演示，实际上您可将纹理从任何带保存功能的图像编辑程序中导入虚幻引擎。

开始之前：须拥有图像编辑程序的访问权。

目标

此指南的要点是为您展示如何从外部图像编辑程序导入纹理。

目的

完成此指南的阅读后，您将了解：

  • 如何将纹理从外部图像编辑程序导入到虚幻编辑器。

从图像编辑程序进行导入

  1. 在图像编辑器中点击 File 和 Save As。

    访问 纹理导入指南 了解关于支持图像格式的更多内容。

  2. 在对话框中，找到 UE4 项目目录和 Content 文件夹（1），输入图像名，选择支持格式（2），然后点击 Save（3）。

  3. 在硬盘上浏览项目的 "Content" 文件夹，找到已添加的纹理。


  4. 如虚幻编辑器打开，将出现一条 "Processing outstanding content changes" 信息。

  5. 虚幻编辑器导入内容之后，即可使用纹理。

    UE4 批处理可为您导入 支持的 图像文件。

    如您看到以下消息弹出：

    UE4 已阻止不支持图像文件的导入。

    如您看到以下消息弹出：

    UE4 告知无法对不规则尺寸的图像进行流或 mipmap 处理。

  6. 未保存的资源以星号标记。

  7. 点击 Save All 按钮保存导入的纹理。

  8. Save Content 对话框出现后，点击 Save Selected 按钮即可保存导入的资源。

  9. 点击 Save Selected 按钮后，星号便会消失，代表资源已成功保存。

确认导入资源

初上手UE4时，最好确认 .uasset文件 已正确导入硬盘。

  1. 在 内容浏览器 中，右键点击 '内容' 文件夹打开 文件夹选项（Folder Options） 对话框（如下图所示）。

  2. 在 文件夹选项（Folder Options） 下，点击 '在浏览器中显示（Show In Explorer）'（如下图所示）。

  3. 浏览器打开后，确认UE4已为您的项目创建相关的 .uasset文件（如下图所示）。
