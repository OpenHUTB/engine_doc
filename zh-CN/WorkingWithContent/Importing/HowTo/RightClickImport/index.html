目标

该简短指南的要点是为您介绍导入内容到虚幻编辑器的多种功能之一。

目的

完成此指南的阅读后，您将了解：

  • 如何使用内容浏览器的 右键单击 导入功能。

使用内容浏览器的右键单击导入功能。

  1. 在内容浏览器的文件框中 单击右键 打开 Import Asset 对话框（点击下图）。

  2. 点击 "Import to /Game..." 打开 Import 对话框。

  3. 找到并选择 Basic_Asset1.fbx、Basic_Asset2.FBX、T_Rock_04_D.TGA 和 T_Rock_04_n.TGA 内容文件。

  4. 点击 Open 开始导入内容文件到项目。

导入内容文件

  1. FBX导入选项（FBX Import Options） 对话框现在将出现在UE4中。点击 导入所有 完成导入进程。

    FBX导入器中有两个导入按钮可用。第一个选择是 导入 按钮，能够以指定设置导入当前选中的FBX文件。第二个选择是 导入所有 按钮，能够以指定设置导入当前选中的所有FBX文件。

    在FBX导入选项参考页面可查看FBX导入器可用设置的详细信息。

  2. 导入进程中，虚幻编辑器右下角将出现一个确认框，显示导入进程的状态。点击 确认 将 T_Rock_04_n.TGA 作为法线贴图导入。

    UE4批导入支持 "点击此处了解支持图像的详情。")的图像文件。

    则说明UE4已阻止不支持图像文件的导入。

    如您看到以下消息弹出：

    则说明UE4告知无法对不规则尺寸的图像进行流送或mipmap处理。

  3. 未保存的资源以星号标记。

  4. 点击 保存所有（Save All） 按钮来保存导入的网格体。

  5. 保存内容（Save Content） 对话框出现后，点击 保存选中项（Save Selected） 按钮即可保存导入的资源。

  6. 点击 保存选中项（Save Selected） 按钮后，星号便会消失，代表资源已成功保存。

确认导入资源

初上手UE4时，最好确认 .uasset文件 已正确导入硬盘。

  1. 在 内容浏览器 中，右键点击 '内容' 文件夹打开 文件夹选项（Folder Options） 对话框（如下图所示）。

  2. 在 文件夹选项（Folder Options） 下，点击 '在浏览器中显示（Show In Explorer）'（如下图所示）。

  3. 浏览器打开后，确认UE4已为您的项目创建相关的 .uasset文件（如下图所示）。
