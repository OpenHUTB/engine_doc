内容示例项目中的 音频 地图是你了解音频系统及其构成的绝佳途径，它展示了多个短小精悍但十分实用的示例。此页面将介绍音频内容示例都包含哪些内容，以及如何查找你要使用的音频系统的具体示例。无论你是初次使用虚幻引擎4中的音频系统，还是想要温习巩固相关内容，此地图都可为你打下良好的基础。

音频地图

通过 音频 地图，你可以了解虚幻引擎4音频的核心概念。此图涵盖了许多主题，包括不同类型的音频资产，以及如何使用它们来按需实现各种效果。无论你是初次使用虚幻引擎4中的音频系统还是想要温习巩固相关内容，本地图都可以为你打下良好的基础。

示例

展示内容

1.1 环境声Actor（AmbientSound Actor）

作为一次性事件播放的音频。

1.2 环境声循环（AmbientSound Looping）

持续播放的音频，直到发出停止播放的指令为止。

1.3 环境声空间化（AmbientSound Spatialize）

自声音位置处播放的音频。

1.4 环境声衰减（AmbientSound Attenuation）

自声音位置处播放并根据距声音的距离而增大或减小音量的音频。

1.5 低通滤波器衰减（Attenuation with Low Pass Filter (LPF)）

自声音位置处播放并随玩家远离声音位置而逐渐变弱的音频。

1.6 衰减形状（Attenuation Shapes）

在指定的形状（如球体、胶囊体、盒体或椎体）内自声音位置播放的音频。

1.7 共享的衰减设置（Shared Attenuation Settings）

展示影响多个声音Actor衰减的音效衰减（Sound Attenuation）资产。

1.8 使用Sound Cue的环境声Actor（AmbientSound Actor using SoundCues）

展示如何使用Sound Cue代替声波（Sound Wave）来创造更为复杂的音效或将多个声音混合在一起。

1.9 音频体积（Audio Volume）

音频受音频体积（Audio Volume）的影响，其效果只有当玩家位于体积内时才能听到（例如，在体积内时，你所听见的对话声就像在礼堂内听到的那样。当你在体积外时，对话正常播放）。

1.10 使用环境区域的音频体积（Audio Volume using Ambient Zone）

受环境区域（Ambient Zone ）影响的两种声音：一种声音被放置在音频体积内，而另一种则不是。体积内的声音在音频体积外时较轻，但在音频体积内时较响；而体积外的声音在音频体积内时较轻，但在音频体积外时较响（例如夜间从室内环境移至室外环境时的音频）。
