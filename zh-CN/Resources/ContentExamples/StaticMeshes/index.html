静态网格体（StaticMeshes） 内容示例地图包含静态网格体使用的重要示例，例如如何设置移动性、指定 UV，以及添加或调整碰撞设置。其他示例还包括：如何使用材质元素自定义静态网格体（可指定应用到静态网格体部分的材质），以及使用 MeshPaint 工具在静态网格体上进行手动绘制。 最后，你可以了解 LOD 如何基于玩家距离相应降低网格体精度，从而提升性能。

静态网格体地图

以下是 静态网格体 地图中提供的示例：

示例

展示内容

1.1 静态网格体（Staticmesh）

在第三方工具中创建并以 FBX 文件格式导入的基础静态网格体。

1.2 移动性（Mobility）

静态网格体移动性分为静态（Static）（例如烘焙光照/阴影，播放时无法移动）和 可移动（Moveable）（动态光照/阴影，播放时可以移动）。

1.3 UV 贴图（UV Mapping）

展示 UV 贴图的正确和错误示例，UV贴图可以定义材质中纹理的对齐和应用至静态网格体的方式。

1.4 光照贴图 UV（Lightmap UVs）

正确与否的光照贴图 UV 示例，定义材质中的纹理在应用到静态网格体后如何响应光照。

1.5 生成独特 UV（Generate Unique UVs）

有三种方法可以生成唯一 UV：通过第三方工具、从通道0自动生成通道1以及自动生成新的通道1。

1.6 材质元素（Material Elements）

说明如何通过材质元素将不同材质指定到静态网格体的不同部分。

1.7 碰撞（Collision）

展示如何对静态网格体应用碰撞。

1.8 插槽（Sockets）

如何将两个静态网格体附加在一个插槽上（例如附加在可旋转转轴上的聚光源）。

1.9 顶点颜色（VertexColors）

通过网格体绘制工具将导入的顶点颜色应用到静态网格体。

1.10 网格体绘制工具（顶点）（MeshPaint Tool (Vertices)）

如何使用网格体绘制工具在静态网格体上绘制颜色。

1.11 网格体绘制工具（纹理）（MeshPaint Tool (Texture)）

如何使用网格体绘制工具在静态网格体上绘制纹理。

1.12 LOD（细节级别）（LOD (Level of Detail)）

展示如何根据玩家距离来改变细节级别。
