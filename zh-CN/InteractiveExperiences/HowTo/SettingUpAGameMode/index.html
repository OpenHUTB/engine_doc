设置游戏模式

关于如何在虚幻引擎 4 中设置游戏模式的操作指南。

Advanced

macOS

本页面的内容

  • 实现指南

GameMode 定义游戏的规则。规则可以包括玩家加入游戏的方式、游戏暂停和关卡过渡以及任何游戏特定的行为，如获胜条件。每个关卡都设置了 GameMode，而且 GameMode 可在多个关卡中重复使用。

实现指南

本页面的内容

  • 实现指南

本指南说明了如何创建 GameMode 蓝图并设置其默认值，如何为游戏分配默认的 GameMode，以及如何借助 World Settings 和 GameMode Override 选项覆盖默认的 GameMode。

创建 Game Mode 蓝图

以下步骤将指引你创建 Game Mode 蓝图并为其设置默认值。

在此示例中，我们使用的是 第三人称模板的蓝图；但是，你可随意使用任何项目。

  1. 在 内容浏览器（Content Browser） 中，点击 新增（Add New） 按钮。

  2. 从下拉菜单的 创建基本资产（Create Basic Asset） 部分中选择 蓝图类（Blueprint Class）。

    可通过 创建高级资产（Create Advanced Asset） 下的 蓝图（Blueprints） 选项来创建各种不同的蓝图资产类型。

  3. 为蓝图资产选择 父类（Parent Class）。欲知更多信息，请参见父类。

在 Pick Parent Class 窗口中选择 Game Mode 类。 在 Pick Parent Class 窗口中，选择 Game Mode Base 类。这是所有 Game Mode 的父类。

编辑 Game Mode 默认设置

  1. Double-click 蓝图将其打开，单击 Class Defaults 按钮以在 Details 面板中打开蓝图默认设置。

  2. Game Mode 下包含若干选项，你可将这些选项设置为游戏默认设置（默认人物、HUD 等）。

    在这里，我们将名为 ThirdPersonCharacter 的人物蓝图分配为玩家在游戏中使用的 Default Pawn Class。

    Game Mode 蓝图指向 Character、HUD、PlayerController、Spectator 和 Game State 类的现有蓝图。你需要分别创建这些蓝图并将其指定为在 Game Mode 蓝图中使用，从而在游戏中实际使用。

分配 Default Game Mode

在上一节中，我们创建了一个 Game Mode 蓝图。拥有 Game Mode 蓝图后，你可以将其分配为游戏中使用的 Default Game Mode。以下步骤将指引你通过 Project Settings 选项分配 Default Game Mode。

  1. 在主编辑器窗口中，单击菜单栏上 Edit 按钮并选择 Project Settings。

  2. 在 Project Settings 窗口中，单击 Maps & Modes 选项。

  3. 在 Default Modes 下的 Maps & Modes 中，单击 Default GameMode 下拉框并根据使用需要分配 GameMode。

    此操作会在加载项目时将你所选的 GameMode 分配为 Default Game Mode。

  4. 如果你单击 Selected GameMode 旁的箭头，你将看到所分配 GameMode 使用的当前设置。

    在这里，我们可以看到 ThirdPersonCharacter 正被用作 Default Pawn Class。

覆写 Default Game Mode

分配 Default Game Mode 后，你可以通过 GameMode Override 部分下的 World Settings 菜单并按关卡将其覆写。以下步骤将向你介绍如何覆写默认 Game Mode。

  1. 在主编辑器窗口中，单击主工具栏上的 World Settings 按钮。

  2. 此操作将打开 World Settings 选项，该选项将在 Details 选项卡所在的右下角窗口中出现。

  3. 在 World Settings 中的 Game Mode 下，你可以单击 GameMode Override 下拉框以更改使用的 GameMode。
