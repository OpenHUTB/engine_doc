按照设计，在 内容浏览器 中创建资源应当快速、便捷并尽可能流畅。只有虚幻需要更多信息时，创建资源才会产生对话框。除此之外，您只需指定名称即可。

添加资源

要创建新资源，可以单击 新建资源（New Asset） 按钮，或在 内容浏览器 空白处 单击右键，然后从快捷菜单中选择所需资源。

删除资源

从虚幻编辑器中删除资源也很简单。只需 右键单击 想要删除的资源，然后从快捷菜单中选择 删除（Delete）。

这样会调出一个确认窗口，要求您确认删除操作，如：

如果您已在关卡中使用该资源，或者有其他资源依赖或引用该资源，则会收到警告窗口，阻止删除操作：

如果是这种情况，您可以选择 替换引用（Replace References） 来选择能取代待删除资源的同类型资源以供引用，也可以选择 强制删除（Force Delete），这样不会取代任何对该资源的引用。这样会使项目中出现引用空资源的情况。

常见操作

当您在 内容浏览器 中 右键单击 资源时会看到一个快捷菜单中，其中 常见（Common） 部分中包含下列选项：

项目

说明

编辑（Edit）

在相应的编辑器中打开所选资源。

重命名（Rename）

允许快速重命名资源。还会更新对该重命名资源的所有引用。

复制（Duplicate）

在当前位置创建所选资源的副本。要将副本挪至他处，将其拖到"源视图"（Source View）中的新文件夹中即可。

选择使用该资源的Actor（Select Actors Using This Asset）

选择关卡中基于该资源的所有Actor。

保存（Save）

将资源保存为._uasset_文件。

删除（Delete）

允许删除，前提是资源没有会阻止删除的仍在生效的引用。

资源操作（Asset Actions）

打开另一个下拉菜单，其中包含更多命令。

资源操作

当您在 内容浏览器 中 右键单击 资源时会看到一个快捷菜单中，其中 资源操作（Asset Actions） 部分中包含下列命令：

项目

说明

使用该项创建蓝图（Create Blueprints Using This）

创建蓝图并将该资源作为组件添加到蓝图。

选择使用该资源的Actor（Select Actors Using This Asset）

选择关卡中基于该资源的所有Actor。

选择包含文件夹（Select Containing Folder）

从"源视图"（Source View）的资源树中选择所选资源所在的文件夹。用于查找位于集合中的资源的真实位置。

在资源管理器中显示（Show In Explorer）

打开其中存储着资源文件._uasset_的Windows资源管理器文件夹。

导出（Export）

允许将资源导出为外部文件格式。可用格式取决于所选资源类型。

迁移（Migrate）

打开迁移工具，可以用来将资源及其所有依赖资源移到新游戏文件夹。有关更多信息，请参阅移植资源。

替换引用（Replace References）

将多个资源合并为一个资源。有关更多信息，请参阅替换引用工具。

属性矩阵（Property Matrix）

打开所选资源的属性矩阵编辑器。有关更多信息，请参阅属性矩阵。

引用处理

在使用 内容浏览器 中的资源时，编辑器会产生一个资源引用列表。了解如何处理这些引用非常重要。如果因为资源正被其他对象引用而无法删除或移除资源，这些工具可以帮助您快速找到引用并将其删除。

当您在 内容浏览器 中 右键单击 资源时会看到一个快捷菜单中，其中 引用（References） 部分中包含下列命令：

项目

说明

复制引用（Copy Reference）

该命令将所有选中对象的引用复制到剪贴板。如果需要将资源路径粘贴到文本字段或者需要一个外部列表时，该命令很有用。

引用查看器（Reference Viewer）

该命令打开引用查看器，其中显示资源及其引用的图表。有关更多信息，请参阅引用查看器。

资源管理

资源和包含资源的文件夹可以从 内容浏览器 中轻松管理。资源可以 在文件夹之间移动或复制，可以创建新文件夹，整个文件夹结构 可以通过_拖放_方法在 资源树 中移动。

移动文件夹

拖放Actor创建

内容浏览器 支持通过拖放操作根据资源在关卡中创建Actor，以及 在文件夹之间移动资源。

将资源拖放到关卡中时，基于该资源的Actor预览将会显示，让您能够看到 该Actor将在关卡中的放置位置和方式。

这些资源按需载入到内存中。

实时渲染缩略图预览

内容浏览器 中可用的许多资源都有可以实时渲染的缩略图。这允许快速预览给定资源，而不必在相应的编辑器中打开。

以下类型的资源可以实时查看：

  • 静态网格体

  • 骨架网格体

  • 材质

  • 包含静态或骨架网格体组件的蓝图
