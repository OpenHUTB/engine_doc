  • 使用方法

全局资源选取器（Ctrl-P）是编辑资源或将资源放入关卡的快速访问方法。它在某些方面类似于 内容浏览器，但工作范围却并不仅限于你在 资源树 中选择的当前文件夹。所以你不仅可以选取特定于你的游戏的资源，还可以选取`引擎`中的任何资源，例如光源或声音发射器。因为它以列表格式提供所有可用资源，所以并不适合作为浏览手段。而如果你知道自己需要的是哪个资源（或资源类型），只想快速在"搜索（Search）"行中输入它的部分名称，那么最好的工具就是它。

将鼠标悬停在任何资源上时，它的所有相关数据都将以 工具提示 的形式显示在窗口底部。

窗口底部的 视图选项（View Options） 与 内容浏览器 上的完全相同，允许你更改它显示和缩放资源的方式。缩略图编辑器在此窗口中不起作用。

当前，你无法在此工具中调出任何资源的 右键单击菜单，但是可以将资源 单击并拖动 到场景中，也可以 双击 资源来调出相关的编辑器。

全局资源选取器（Global Asset Picker） 窗口不可移动，也不可调整大小，而且只要你在窗口以外点击、将资源拖动到场景中或 双击 编辑资源，该窗口就会立即关闭。

使用方法

查看 全局资源选取器 非常简单，只需按下 Ctrl + P，然后搜索需要的资源即可。

然后你可以将资源 单击并拖动 到场景中，也可以 双击 资源来调出相关的编辑器。
