虚幻编辑器（Unreal Editor） 是一套编辑器、浏览器和工具，用于在虚幻引擎中处理内容。其核心作用是 设计关卡；但其中包含用于为您的项目导入和操纵内容的编辑器和浏览器。

界面元素类型

虚幻编辑器的界面在布局上是完全可配置的。它由包含选项卡的窗口组成，这些窗口可以根据您的需要重新排列。这些窗口可以是顶级编辑器，也可以是这些编辑器中的选项卡。

编辑器

编辑器（Editors） 类似于虚幻编辑器中的应用程序，例如关卡编辑器和蓝图编辑器。 它们针对开发或设计中过程的特定方面。有些编辑器，在任何时候只能有一个实例存在，比如关卡编辑器；而其他编辑器，可以创建多个实例，但是在任何时候一个编辑器只有一个实例可以编辑特定的资源（比如蓝图编辑器）。

一般编辑器通过 窗口（Window） 菜单打开，而资源编辑器通过选择编辑特定类型的资源打开。关卡编辑器（Level Editor） 是一种特殊情况，它总是在虚幻编辑器打开时出现。

编辑器可以通过单击编辑器的选项卡上的 关闭应用程序（Close App）（X）按钮来关闭，或通过 右键单击（right-clicking） 并选择 关闭（Close） 来关闭。

每个编辑器由一个 菜单栏（Menu Bar） 和一个或多个选项卡（Tab）组成，它们构成编辑器的大部分功能。

编辑器选项卡

如果您同时使用多个编辑器，那么将它们停靠在一起可能有助于节省屏幕空间。每个编辑器的顶部都有一个斜面选项卡，其中包含编辑器的名称和正在编辑的资源。通过拖动该选项卡，可以将该编辑器与视图中的任何其他编辑器停靠在一起。

编辑器选项卡与面板上使用的选项卡不同。面板可以停靠到其原生编辑器中的任何位置。编辑器选项卡只能与其他编辑器停靠在一起。此外，请注意形状上的差异；编辑器选项卡在两侧倾斜，而面板选项卡则保持正方。

面板

面板（Panels） 是编辑器的专门部分，它们提供编辑器在特定方面的功能。编辑器可以包含任意数量的面板。属于特定编辑器的面板只影响其父编辑器，并且只能在该特定编辑器内停靠。 面板同样通过 窗口（Window） 菜单打开。

大部分面板的左上角都有选项卡。这些选项卡可用于重定位、重停靠或关闭面板。您可以通过 右键单击 选项卡并选择 关闭（Close） 或单击选项卡右侧的小 X 来关闭面板。

只要可能，选项卡都将以当前在该选项卡的编辑器中编辑的资源命名。例如，关卡编辑器（Level Editor）选项卡将显示当前正在编辑的关卡的名称。

如果某个选项卡是一个选项卡井中唯一的选项卡，您可以通过 右键单击 它并选择 隐藏选项卡（Hide Tabs） 来隐藏它。或者，您也可使用 选项卡选项（Tab Options） 菜单中的 隐藏选项卡（Hide Tabs） 选项来隐藏它。这样就可以很好地折叠选项卡， 从而只显示选项卡的内容部分。当某个标签被隐藏时，左上角变成金色。

单击左上角将再次展开该选项卡。

菜单

菜单将位于任何相应编辑器窗口的顶部。有些菜单，例如 窗口（Window） 和 帮助（Help），在所有编辑器窗口中都是一致的。

停靠和布局

您只需将选项卡从当前位置拖动到要停靠的位置，即可停靠选项卡。

编辑器中选项卡（包括浮动窗口）的布局在会话之间是持久性的，并在关闭编辑器时自动保存。选项卡还会记住它们的最后一个位置，并在那里重新打开。

保存与加载自定义布局

虚幻编辑器现在具有创建、保存、加载并导入多个虚幻编辑器UI布局的功能（甚至可以跨机器或设备）。个人用户不但可以创建和保存自定义布局，团队成员也可以创建特定的虚幻编辑器布局来改进工作流程，并将这些布局分享给团队中其他成员。

在窗口菜单的底部，有一个名为 布局（Layout） 的分段。在布局分段中，有一些选项可用于 加载布局（Load Layout）、保存布局（Save Layout） 和 移除布局（Remove Layout）。

在 编辑器偏好设置（Editor Preferences） 中，您可以禁用或启用 编辑器布局管理（Editor Layout Management）。在打开的项目中，点击 编辑（Edit）>编辑器偏好设置（Editor Preferences） 将其打开，然后进入 通用（General）> 外观（Appearance）> 用户界面（User Interface） 并点击 启用编辑器布局管理（Enable User Editor Layout Management) 旁的复选框来启用或禁用该功能。禁用编辑器布局管理将移除 保存布局（Save Layout） 和 移除布局（Remove Layout） 菜单，同时移除子菜单选项 导入布局（Import Layout）。

加载布局

本菜单具有以下子菜单选项：

  • 默认编辑器布局（Default Editor Layout）：加载默认布局。

  • 用户布局（User Layouts）：加载用户创建或导入的布局。该选项只有在您拥有可用的保存或导入布局时才会显示。

  • 导入布局（Import Layout）: 从计算机上的任何位置导入一个（或多个）布局并立即加载它。这些导入的布局将被保存并显示在用户布局分段中，可随时加载。

保存布局

本菜单具有以下子菜单选项：

  • 将布局另存为（Save Layout As）：使用自定义名称保存布局。它将被保存到磁盘，并自动显示在用户布局分段中。

  • 导出布局（Export Layout）：与**将布局另存为（Save Layout As）** 类似，但它可以被保存早计算机上的任何布鲁中，而且不会显示在用户布局分段中。这对于跨机器导出/导入非常有用。

移除布局

本菜单具有以下子菜单选项：

  • 用户布局（User Layouts）：显示您所保存或导入的所有布局。

  • 移除所有用户布局（Remove All User Layouts）：移除用户布局目录中保存的所有布局。
