设置适用于虚幻引擎的Shotgun项目

描述如何使用设置现有Shotgun项目配置，使其适用于虚幻工具集引擎。

设置Shotgun项目以与虚幻引擎4配合工作的流程与内容流程中的其他应用程序（例如，Autodesk Maya或3ds Max、或Foundry Nuke）的流程相似。您将需要将Shotgun项目配置为使用 工具包引擎，它是一系列工具和脚本，旨在让Shotgun项目了解虚幻引擎并将Shotgun工具和工作流集成到虚幻编辑器界面。

针对每个要与虚幻引擎配合使用的Shotgun项目，只需进行一次本页面中列出来的配置。此配置通常由Shotgun管理员完成。如果您已设置过Shotgun项目并将它们配置为与其他内容创建工具集成，那么此流程对您来说应该相当熟悉了。如果对此流程不太熟悉，请参阅以下资源：

  • 集成用户手册总体概述了如何快速上手使用Shotgun集成。

  • 集成管理员手册对流程配置进行了详细说明。

开始之前

我们建议您使用GitHub上tk-config-unreal元库中已为您设置好的项目配置。此配置对基本Shotgun工具包配置进行了扩展，增加了构成虚幻集成的元素。

  • 要使用GitHub中的这一配置，计算机上必须安装有Git。如果尚未安装Git，请从此处下载。

1 - 将虚幻引擎软件添加到Shotgun

在此步骤中，您需要将虚幻引擎集成作为新软件添加到您的组织。

  1. 在网页浏览器中登录您所在组织的Shotgun页面。然后在用户菜单中选择 Software。

    点击查看全图

  2. 在 Software 页面上点击 + Software 按钮。

  3. 将 Software Name 设置为 Unreal Engine，并将 Engine 设置为 tk-unreal。 

  4. 点击 Create Software。
    在返回 Software 页面后，您将可以在列表中看到新添加的虚幻引擎条目。

最终结果

Shotgun现在已经得知了虚幻引擎集成的存在，您可以继续按照以下步骤将组织中的Shotgun项目与虚幻编辑器配合使用。

2 - 设置Shotgun项目

在此步骤中，您可以设置一个现有的Shotgun项目，使其能够使用虚幻引擎集成。

  1. 打开Shotgun Desktop应用程序，从 项目（Projects） 页面中选择要与虚幻引擎配合使用的项目。

  2. 单击您的用户图片，并从菜单中选择 高级项目设置...（Advanced project setup...）。


    单击查看大图。

  3. 选择 Git 配置选项并单击 继续（Continue）。


    单击查看大图。

  4. 将虚幻配置元库的URL
    https://github.com/ue4plugins/tk-config-unreal.git
    粘贴进去并单击 继续（Continue）。


    单击查看大图。

  5. 根据 Shotgun设置项目向导 中的剩余提示进行操作，为Shotgun项目设置共享的存储位置并为流程配置设置位置。

最终结果

项目设置完成后，Shotgun Desktop将扫描您的计算机以查找虚幻引擎安装并在项目的"应用程序（Apps）"页面上将它们列出：

如果从其中一个快捷方式启动虚幻并打开一个启用了Shotgun插件的虚幻引擎项目，您将能够如将虚幻引擎与Autodesk Shotgun配合使用页面中所述使用Shotgun集成。
